# ClaimLens
Claimlens is a production ready, end to end machine learning systems that classifies tiktok videos as claim or opinion using video description, auther metadata, and engagement signals. The project is designed with real world machine learning practice, covering data validation, feature pipeline, training pipeline, hyperparameters tuning, experiment tracking, ci and api based inference.

The repository emphasizes scalability, reproducibility, and deplyment ready, making it suitable for content moderation, misinformation research, and social media intelligence use case.
